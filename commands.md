### List of apache2 commands and notes

#### Commands:
```
$ sudo cat /var/log/apache2/error.log
```
Displays error log
```
$ sudo tail /var/log/apache2/access.log -f
```
Displays access log
```
$ sudo service apache restart
```
Restarts the server

#### Notes:
- `var/` is stored in root (accessed by `cd /`)
